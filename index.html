Eclairs Makes Your Websites 10x Faster

Eclairs provides pre made components that you can directly use in your websites without any hassle. We did all the hardwork so you don't have to.

Let's Get Started

Install the Eclairs UI package and its dependencies using npm or yarn.

Wrap your app in a ThemeProvider at its root for styling.

Eclairs is open-source.
